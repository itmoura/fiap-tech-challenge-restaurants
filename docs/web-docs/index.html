Sobre o Projeto

O Tech Challenge - Restaurant Management API é um sistema completo de gerenciamento de restaurantes desenvolvido com as melhores práticas de desenvolvimento moderno. Utilizando Spring Boot e MongoDB, o sistema oferece alta performance e escalabilidade.

🎯 Objetivos Principais

  • Performance: Otimização para leitura com documentos aninhados
  • Escalabilidade: Arquitetura preparada para crescimento
  • Manutenibilidade: Código limpo e bem estruturado
  • Testabilidade: Cobertura completa de testes
⚡

Quick Start

Execute o projeto em poucos passos:

# 1. Clone o repositório
git clone https://github.com/itmoura/fiap-tech-challenge-restaurants.git
cd fiap-tech-challenge-restaurants

# 2. Inicie o MongoDB
docker-compose up -d mongodb

# 3. Execute a aplicação
./gradlew bootRun

# 4. Teste a API
curl http://localhost:8081/api/restaurants

🎉 Pronto! A API estará disponível em http://localhost:8081
