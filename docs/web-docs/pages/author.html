Sobre o Tech Challenge

O Tech Challenge - Restaurant Management API foi desenvolvido como projeto final da pós-graduação em Arquitetura de Software da FIAP, demonstrando a aplicação prática de conceitos avançados de desenvolvimento.

🎯 Objetivos do Projeto

  • Aplicar Clean Architecture: Demonstrar separação clara de responsabilidades
  • Implementar DDD: Modelagem focada no domínio do negócio
  • Usar NoSQL eficientemente: Aproveitar as vantagens do MongoDB
  • Criar API RESTful: Seguir padrões e boas práticas
  • Documentar completamente: Facilitar uso e manutenção
  • Implementar testes: Garantir qualidade e confiabilidade
