🏪 Restaurants API

CRUD completo para gerenciamento de restaurantes

📋

Sobre Restaurants

A API de Restaurants é o núcleo do sistema, permitindo o gerenciamento completo de restaurantes com suas informações básicas, endereços e menus estruturados.

🏷️

Informações Básicas

Nome, descrição e tipo de cozinha

📍

Endereço Completo

Endereço estruturado com validação

🍽️

Menu Integrado

Menu com categorias e itens aninhados

🔍

Busca Avançada

Filtros por tipo, localização e nome

🔗

Endpoints

GET /api/restaurants Listar todos os restaurantes
GET /api/restaurants/{id} Buscar restaurante por ID
GET /api/restaurants/search Buscar restaurantes com filtros
POST /api/restaurants Criar novo restaurante
PUT /api/restaurants/{id} Atualizar restaurante
DELETE /api/restaurants/{id} Remover restaurante
📊

Modelo de Dados

🏗️ Estrutura Restaurant

{
  "id": "string",                    // ID único gerado automaticamente
  "name": "string",                  // Nome do restaurante
  "description": "string",           // Descrição detalhada
  "kitchenType": "string",          // Tipo de cozinha (enum)
  "address": {                      // Endereço estruturado
    "street": "string",             // Rua e número
    "city": "string",               // Cidade
    "state": "string",              // Estado (UF)
    "zipCode": "string"             // CEP
  },
  "menu": {                         // Menu do restaurante
    "categories": [                 // Array de categorias
      {
        "id": "string",
        "name": "string",
        "description": "string",
        "items": [                  // Array de itens
          {
            "id": "string",
            "name": "string",
            "description": "string",
            "price": "number"
          }
        ]
      }
    ]
  }
}

✅ Validações

Campo Regras Exemplo
name Obrigatório, 2-100 caracteres Restaurante Italiano
description Opcional, máximo 500 caracteres Autêntica culinária italiana
kitchenType Obrigatório, enum válido ITALIAN
address.street Obrigatório, 5-200 caracteres Rua Augusta, 123
address.zipCode Obrigatório, formato 00000-000 01234-567
💻

Exemplos de Uso

📋 Listar Restaurantes

curl -X GET http://localhost:8081/api/restaurants \
  -H "Content-Type: application/json"

Resposta:

{
  "success": true,
  "data": [
    {
      "id": "64f1a2b3c4d5e6f7g8h9i0j1",
      "name": "Bella Italia",
      "description": "Autêntica culinária italiana",
      "kitchenType": "ITALIAN",
      "address": {
        "street": "Rua Augusta, 123",
        "city": "São Paulo",
        "state": "SP",
        "zipCode": "01234-567"
      }
    }
  ],
  "timestamp": "2024-08-05T19:30:00Z"
}

🔍 Buscar por ID

curl -X GET http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1 \
  -H "Content-Type: application/json"

🔍 Busca com Filtros

curl -X GET "http://localhost:8081/api/restaurants/search?kitchenType=ITALIAN&city=São Paulo" \
  -H "Content-Type: application/json"

➕ Criar Restaurante

curl -X POST http://localhost:8081/api/restaurants \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Novo Restaurante",
    "description": "Descrição do restaurante",
    "kitchenType": "BRAZILIAN",
    "address": {
      "street": "Rua Nova, 456",
      "city": "São Paulo",
      "state": "SP",
      "zipCode": "01234-567"
    }
  }'

✏️ Atualizar Restaurante

curl -X PUT http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Bella Italia Renovado",
    "description": "Renovado com novo chef",
    "kitchenType": "ITALIAN",
    "address": {
      "street": "Rua Augusta, 123",
      "city": "São Paulo", 
      "state": "SP",
      "zipCode": "01234-567"
    }
  }'

🗑️ Remover Restaurante

curl -X DELETE http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1 \
  -H "Content-Type: application/json"
🔍

Parâmetros de Busca

O endpoint /api/restaurants/search suporta os seguintes parâmetros:

Parâmetro Tipo Descrição Exemplo
name string Busca por nome (parcial) ?name=Bella
kitchenType enum Filtro por tipo de cozinha ?kitchenType=ITALIAN
city string Filtro por cidade ?city=São Paulo
state string Filtro por estado ?state=SP
page integer Número da página (0-based) ?page=0
size integer Tamanho da página (max 100) ?size=20

🔗 Exemplos de Busca Combinada

# Restaurantes italianos em São Paulo
GET /api/restaurants/search?kitchenType=ITALIAN&city=São Paulo

# Busca por nome com paginação
GET /api/restaurants/search?name=Bella&page=0&size=10

# Múltiplos filtros
GET /api/restaurants/search?kitchenType=BRAZILIAN&state=SP&page=0&size=20
