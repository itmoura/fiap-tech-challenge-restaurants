👨‍🍳 Kitchen Types API

Gerenciamento de tipos de cozinha (Italiana, Brasileira, etc.)

📋

Sobre Kitchen Types

Os Kitchen Types representam os diferentes tipos de cozinha disponíveis no sistema (Italiana, Brasileira, Japonesa, etc.). Cada restaurante deve ter um tipo de cozinha associado.

🏷️

Identificação

Cada tipo tem um nome único e descrição

🔗

Relacionamento

Associado a restaurantes via campo kitchenType

📊

Enum Values

Valores pré-definidos para consistência

🍽️

Tipos Disponíveis

Código Nome Descrição
ITALIAN Italiana Cozinha tradicional italiana com massas, pizzas e risotos
BRAZILIAN Brasileira Culinária brasileira com pratos regionais e tradicionais
JAPANESE Japonesa Cozinha japonesa com sushi, sashimi e pratos orientais
MEXICAN Mexicana Culinária mexicana com tacos, burritos e especiarias
CHINESE Chinesa Cozinha chinesa tradicional com wok e dim sum
FRENCH Francesa Alta gastronomia francesa com técnicas refinadas
🔗

Endpoints

GET /api/kitchen-types Listar todos os tipos de cozinha
GET /api/kitchen-types/{id} Buscar tipo de cozinha por ID
POST /api/kitchen-types Criar novo tipo de cozinha
PUT /api/kitchen-types/{id} Atualizar tipo de cozinha
DELETE /api/kitchen-types/{id} Remover tipo de cozinha
💻

Exemplos de Uso

📋 Listar Tipos de Cozinha

curl -X GET http://localhost:8081/api/kitchen-types \
  -H "Content-Type: application/json"

Resposta:

{
  "success": true,
  "data": [
    {
      "id": "64f1a2b3c4d5e6f7g8h9i0j1",
      "name": "ITALIAN",
      "description": "Cozinha tradicional italiana"
    },
    {
      "id": "64f1a2b3c4d5e6f7g8h9i0j2",
      "name": "BRAZILIAN", 
      "description": "Culinária brasileira"
    }
  ],
  "timestamp": "2024-08-05T19:30:00Z"
}

🔍 Buscar por ID

curl -X GET http://localhost:8081/api/kitchen-types/64f1a2b3c4d5e6f7g8h9i0j1 \
  -H "Content-Type: application/json"

Resposta:

{
  "success": true,
  "data": {
    "id": "64f1a2b3c4d5e6f7g8h9i0j1",
    "name": "ITALIAN",
    "description": "Cozinha tradicional italiana com massas, pizzas e risotos"
  },
  "timestamp": "2024-08-05T19:30:00Z"
}

➕ Criar Tipo de Cozinha

curl -X POST http://localhost:8081/api/kitchen-types \
  -H "Content-Type: application/json" \
  -d '{
    "name": "THAI",
    "description": "Cozinha tailandesa com especiarias e sabores únicos"
  }'

Resposta:

{
  "success": true,
  "data": {
    "id": "64f1a2b3c4d5e6f7g8h9i0j3",
    "name": "THAI",
    "description": "Cozinha tailandesa com especiarias e sabores únicos"
  },
  "message": "Tipo de cozinha criado com sucesso",
  "timestamp": "2024-08-05T19:30:00Z"
}

✏️ Atualizar Tipo de Cozinha

curl -X PUT http://localhost:8081/api/kitchen-types/64f1a2b3c4d5e6f7g8h9i0j3 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "THAI",
    "description": "Autêntica cozinha tailandesa com curry e pad thai"
  }'

🗑️ Remover Tipo de Cozinha

curl -X DELETE http://localhost:8081/api/kitchen-types/64f1a2b3c4d5e6f7g8h9i0j3 \
  -H "Content-Type: application/json"

Resposta:

{
  "success": true,
  "message": "Tipo de cozinha removido com sucesso",
  "timestamp": "2024-08-05T19:30:00Z"
}
📊

Modelo de Dados

🏗️ Estrutura KitchenType

{
  "id": "string",           // ID único gerado automaticamente
  "name": "string",         // Nome do tipo (enum: ITALIAN, BRAZILIAN, etc.)
  "description": "string"   // Descrição detalhada do tipo de cozinha
}

✅ Validações

Campo Regras Exemplo
name Obrigatório, único, enum válido ITALIAN
description Obrigatório, 10-200 caracteres Cozinha tradicional italiana

🔗 Relacionamentos

  • Restaurants: Um tipo de cozinha pode ter vários restaurantes
  • Constraint: Não pode ser removido se houver restaurantes associados
⚠️

Tratamento de Erros

❌ Erros Comuns

400 - Bad Request

{
  "success": false,
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Dados inválidos",
    "details": "Nome é obrigatório"
  },
  "timestamp": "2024-08-05T19:30:00Z"
}

404 - Not Found

{
  "success": false,
  "error": {
    "code": "KITCHEN_TYPE_NOT_FOUND",
    "message": "Tipo de cozinha não encontrado",
    "details": "Nenhum tipo de cozinha encontrado com o ID fornecido"
  },
  "timestamp": "2024-08-05T19:30:00Z"
}

409 - Conflict

{
  "success": false,
  "error": {
    "code": "KITCHEN_TYPE_ALREADY_EXISTS",
    "message": "Tipo de cozinha já existe",
    "details": "Já existe um tipo de cozinha com este nome"
  },
  "timestamp": "2024-08-05T19:30:00Z"
}
