🍕 Menu Items API

Gerenciamento de itens específicos do menu com preços e descrições

📋

Sobre Menu Items

Os Menu Items são os pratos específicos oferecidos pelo restaurante, organizados dentro de categorias. Cada item possui nome, descrição, preço e pertence a uma categoria específica.

🍽️

Pratos Específicos

Cada item representa um prato do menu

💰

Preços

Cada item tem seu preço específico

📝

Descrições

Descrições detalhadas dos ingredientes

📂

Categorizado

Pertence a uma categoria específica

🔗

Endpoints

GET /api/restaurants/{id}/menu/categories/{categoryId}/items Listar itens da categoria
GET /api/restaurants/{id}/menu/categories/{categoryId}/items/{itemId} Buscar item específico
POST /api/restaurants/{id}/menu/categories/{categoryId}/items Adicionar item à categoria
PUT /api/restaurants/{id}/menu/categories/{categoryId}/items/{itemId} Atualizar item
DELETE /api/restaurants/{id}/menu/categories/{categoryId}/items/{itemId} Remover item
📊

Modelo de Dados

🏗️ Estrutura MenuItem

{
  "id": "string",           // ID único do item
  "name": "string",         // Nome do prato
  "description": "string",  // Descrição detalhada
  "price": "number"         // Preço em reais (formato decimal)
}

✅ Validações

Campo Regras Exemplo
name Obrigatório, 2-100 caracteres Spaghetti Carbonara
description Opcional, máximo 300 caracteres Massa com bacon, ovos e queijo parmesão
price Obrigatório, > 0, máximo 2 decimais 32.90

💰 Formato de Preço

  • Moeda: Real brasileiro (BRL)
  • Formato: Decimal com até 2 casas
  • Mínimo: 0.01 (1 centavo)
  • Máximo: 9999.99
  • Exemplos válidos: 15.50, 32.90, 125.00
💻

Exemplos de Uso

📋 Listar Itens da Categoria

curl -X GET http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1/menu/categories/cat1/items \
  -H "Content-Type: application/json"

Resposta:

{
  "success": true,
  "data": [
    {
      "id": "item1",
      "name": "Bruschetta Clássica",
      "description": "Pão italiano com tomate, manjericão e azeite",
      "price": 18.90
    },
    {
      "id": "item2",
      "name": "Carpaccio de Salmão",
      "description": "Fatias finas de salmão com alcaparras e limão",
      "price": 28.50
    }
  ],
  "timestamp": "2024-08-05T19:30:00Z"
}

🔍 Buscar Item Específico

curl -X GET http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1/menu/categories/cat1/items/item1 \
  -H "Content-Type: application/json"

➕ Adicionar Item

curl -X POST http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1/menu/categories/cat1/items \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Salada Caesar",
    "description": "Alface romana, croutons, parmesão e molho caesar",
    "price": 22.90
  }'

Resposta:

{
  "success": true,
  "data": {
    "id": "item3",
    "name": "Salada Caesar",
    "description": "Alface romana, croutons, parmesão e molho caesar",
    "price": 22.90
  },
  "message": "Item adicionado com sucesso",
  "timestamp": "2024-08-05T19:30:00Z"
}

✏️ Atualizar Item

curl -X PUT http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1/menu/categories/cat1/items/item3 \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Salada Caesar Premium",
    "description": "Alface romana, croutons artesanais, parmesão 24 meses e molho caesar da casa",
    "price": 26.90
  }'

🗑️ Remover Item

curl -X DELETE http://localhost:8081/api/restaurants/64f1a2b3c4d5e6f7g8h9i0j1/menu/categories/cat1/items/item3 \
  -H "Content-Type: application/json"

Regras de Negócio

📂

Categoria Existente

Item só pode ser criado em categoria existente

🏷️

Nome Único

Não pode haver itens com mesmo nome na categoria

💰

Preço Válido

Preço deve ser positivo com máximo 2 decimais

🔄

Operação Atômica

Todas as operações são atômicas no documento

⚠️ Validações Especiais

  • Restaurante não encontrado: Retorna 404
  • Categoria não encontrada: Retorna 404
  • Item duplicado na categoria: Retorna 409 Conflict
  • Item não encontrado: Retorna 404
  • Preço inválido: Retorna 400 Bad Request
  • Nome muito longo: Retorna 400 Bad Request
⚠️

Tratamento de Erros

❌ Erros Específicos

{
  "success": false,
  "error": {
    "code": "MENU_ITEM_NOT_FOUND",

  },

{
  "success": false,
  "error": {
    "code": "MENU_ITEM_ALREADY_EXISTS",
    "message": "Item já existe",
    "details": "Já existe um item com este nome na categoria",

  "timestamp": "2024-08-05T19:30:00Z"
}

{
  "success": false,
  "error": {
    "code": "INVALID_PRICE",

    "value": -10.50
  },
  "timestamp": "2024-08-05T19:30:00Z"
}
